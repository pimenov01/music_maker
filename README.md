# DND Music Generator

Атмосферный генератор звуков для настольных ролевых игр в мрачной sci-fi / grimdark эстетике.

Проект позволяет запускать несколько звуковых слоёв одновременно: эмбиент, боевые треки, погодные эффекты и отдельные звуковые события. Каждый звук зациклен, плавно появляется и исчезает через fade-in / fade-out, а громкость каждого слоя регулируется отдельно.
